# Exemples à ne pas reproduire

Mettre ici une ou plusieurs images de **mauvais** rendus (mauvaise pliure, mauvais alignement, cadrage, ombres). Une image est choisie aléatoirement et envoyée à Gemini avec la consigne de ne pas reproduire ce rendu.

**Exemple** : copier une mauvaise sortie 1_face ou 2_back, par ex. :
```bash
cp output_batch_stone_island/1200005-V05G/1_face.png refs/errors/bad/1200005-V05G-bad-fold.png
```

Nom libre (`.png` ou `.jpg`). Le CLI ignore ce README.
